webpage name: DigiTools


a description for the webpage:
A digital tools buying website where you can browse and buy premium tools like AI writing tools, design templates, stock assets and more. I built this as part of an assignment project using React and Tailwind CSS.


Technologies Used:

React.js
Tailwind CSS
DaisyUI
JavaScript
React-Toastify

 Features

 You can add products to cart and remove them individually or clear the whole cart when you proceed to checkout
 Toast notifications pop up whenever you add or remove a product so you always know whats happening
 The site is fully responsive so it works on mobile and desktop both
